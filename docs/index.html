Owner signups may require a referral code. If you need one, contact Foodtruckfinderinfo@gmail.com.

Yes. Owners can share live truck location updates so customers can find the truck more easily.

Can I add my menu and deals?

Yes. The owner tools support menu updates, truck details, and limited-time specials.

Is this just for truck owners?

No. The app also supports foodies and organizers, but this page is built for truck owners deciding whether to join.
