# Demo Video Brief

Put the homepage demo video in this folder as:

- `ftf-demo.mp4`

The root website automatically looks for that file at:

- `assets/media/video/ftf-demo.mp4`

Current generated file:

- `ftf-demo.mp4`

To rebuild it from the current screenshot set:

```bash
./scripts/build-demo-video.sh
```

That script uses:

- `scripts/render_demo_slides.py` to create branded slide images
- `ffmpeg` to encode the final MP4

## Recommended video structure

1. Open with the foodie value:
   Show the home or explore flow and answer the question "How do I find a truck fast?"
2. Move to owner value:
   Show the owner dashboard, schedule updates, menu updates, and profile editing.
3. End with organizer value:
   Show truck discovery, invite-oriented screens, and explain event coordination.
4. Close with one direct CTA:
   `Download Food Truck Finder` or `Contact us for owner onboarding`.

## Recommended length

- `45-60 seconds` for the homepage
- `15-20 seconds` cutdowns for social posts

## Suggested shot order

1. `foodies/explore-live-simulator.png` as the poster frame
2. Quick browse through nearby trucks
3. Menu and truck details
4. Owner dashboard
5. Schedule update
6. Menu update
7. Organizer invite flow
8. Download CTA

## Recording notes

- Use vertical or square crops only for social.
- Use a landscape or clean centered mobile capture for the website.
- Keep labels minimal and benefit-focused.
