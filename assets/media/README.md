# Website Media Library

This folder contains the screenshots selected for the public website and grouped by audience so the page can stay organized.

## Structure

- `foodies/`: discovery, home feed, and explore screens
- `owners/`: owner dashboard, schedule, menu, profile setup, and annotated owner assets
- `organizers/`: role-based signup, discovery, and invite-oriented visuals
- `video/`: homepage demo video slot and recording brief
